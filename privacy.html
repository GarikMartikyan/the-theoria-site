Privacy Policy

Last updated: July 2026

This Privacy Policy explains how The Theoria ("we", "us") handles information in connection with this website and any API integrations we operate.

This website

This is a static informational site. It does not use cookies, does not run analytics, and does not collect any personal information from visitors.

Platform API integrations

We use official platform APIs (YouTube Data API, Instagram Graph API, TikTok Content Posting API) solely to publish videos we produce to The Theoria's own accounts on those platforms. These integrations:

  • Only ever authenticate as The Theoria's own accounts — not on behalf of, or with access to, any other person's account or data.
  • Do not collect, store, or process personal data belonging to viewers, followers, or any other platform users.
  • Only transmit the video file and its metadata (title, description, hashtags) that we author ourselves for publishing.

Data we store

Access credentials required to publish to our own accounts are stored privately and are never shared with third parties. We do not sell or share any data with advertisers or data brokers.

Changes

We may update this Privacy Policy from time to time; the "Last updated" date above will reflect the most recent revision.

Contact

Questions about this policy can be sent via our social media channels linked on the homepage.
